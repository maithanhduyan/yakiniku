# DASHBOARD ADMIN GUIDE

Màn hình `Dashboard` là trung tâm quản lý các thiết bị trong nhà hàng (kitchen,table-order,pos,checkin)
Dashboard có quyền cho phép thiết bị nào được phép login vào hệ thống
Hãy thêm chức năng quản lý thiết bị: tạo mã QR Code để cho phép thiết bị đăng nhập.
ví dụ: cấp phép cho table-order: trước khi tạo phải có đủ thông tin. branch,table name/code --> QR Code)
